# Surface Pro 6 Hackintosh

# Specs:
- CPU: Intel i5-8350U
- GPU: Intel UHD Graphics 620
- RAM: 8GB LPDDR3 1866 MHz

# Working:
- Intel HD Graphics 
- Restart and Shutdown
- CPU Power Management
- Audio
- Touchpad
- All USB Ports

# Not Working:
- Wifi/ Bluetooth/ Airdrop
- Camera
